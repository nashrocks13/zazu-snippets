## Snippet manager for Zazu

## Usage

Finding a snippet:

~~~
snip emoji
~~~

Creating a snippet from your clipboard:

~~~
snipc emoji
~~~

Delete a snippet by name:

~~~
snipd emoji
~~~

## Installing

Add the package to your plugins array in `./zazurc.json`.

~~~ json
{
  "plugins": [
    "nashrocks13/zazu-snippets"
  ]
}
~~~
